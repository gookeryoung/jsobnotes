# ts.变量.声明

`typescript` 采取以下方式声明变量及初始值：

```typescript
let [变量名]: [类型] = 值;
```

例如:

```typescript
let nickname: string = 'Jacky';  
let age: number = 23;  
const sum: number = 123.18;  
console.log(`${nickname}, ${age}, ${sum}`);
// Jacky, 23, 123.18
```
