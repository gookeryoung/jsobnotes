# 函数声明

函数声明或函数定义以 `function` 关键字开头，后面跟函数的名称。函数所包含的 javascript 语句在一对花括号中进行定义。

注意：函数声明的作用域会提升到文件作用域的顶部，也就意味着在声明以前也可以调用。

## 示例

常规调用:

```javascript
function logCompliment() {
	console.log('You are doing good!')
}

logCompliment();
// You are doing good!
```

声明前调用:

```javascript
logCompliment();
// You are doing good!

function logCompliment() {
	console.log('You are doing good!')
}
```
