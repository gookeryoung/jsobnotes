# ts.类型.null

在 `javascipt` 中 `null` 表示什么都没有。它是一个只有一个值的特殊类型，表示一个空对象应用。

注意：用 typeof 检测 null 返回的是 object。

## 示例

```typescript
const fun = null;
console.log(`fun = ${fun}, type of fun is ${typeof fun}`);
// fun = null, type of fun is object
```
