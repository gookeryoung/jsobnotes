# 函数默认参数

可以为函数提供默认参数值，默认参数可以是任何类型，不限于字符串。

## 示例

```javascript
function buy(name = 'Jack', products = ['chicken', 'egg']) {
	console.log(`${name} bought ${products}.`);
}

console.log(buy());
// Jack bought chicken,egg.
```
